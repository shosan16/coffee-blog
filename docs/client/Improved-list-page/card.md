## カード仕様

### タイトル

- 2行固定（min-height / max-height で固定）
- 溢れた場合は省略記号（-webkit-line-clamp: 2）

### 焙煎度表示

- **左サイドライン**: カード左端に6pxの縦線（焙煎度に応じた色）
- **右上バッジ**: 小さいラベルで焙煎度テキスト表示
- 6段階の焙煎度と色:
  - 浅煎り: #e2b714
  - 中浅煎り: #c9a227
  - 中煎り: #a67c52
  - 中深煎り: #7a5a3a
  - 深煎り: #5c4033
  - 極深煎り: #2d1f1a

### タグ表示

- 2行分の高さを確保（max-height: 56px）
- **2行に収まる場合**: 全タグを表示
- **2行を超える場合**: 2行分表示 + 右下に「他N件 ▼」ボタン
- 「他N件」クリックでドロップダウン表示（残りタグを表示）
- タグクリックでフィルタリング機能

### 投稿者名

- タグエリアの下に配置
- 区切り線（border-top）で分離
- 左揃え、グレー系の控えめな色

### インタラクション

- カードホバー: 影 + 少し浮き上がる
- タグホバー: 背景色反転
- タグクリック: フィルターに追加

## デザイントーン

- 背景色: #f7f5f2（温かみのあるオフホワイト）
- カード背景: #ffffff
- テキスト: #2c2420（ダークブラウン）
- フォント: Zen Kaku Gothic New（Google Fonts）
