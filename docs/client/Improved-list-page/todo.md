# RecipeCard UI改善 - タスクリスト

## 概要

レシピ一覧ページのカードUIを刷新し、デザイントーンを統一する。

---

## Phase 1: RecipeCard UI改善（ダミーデータ使用）

- [ ] RecipeCard コンポーネントの刷新
  - 左サイドライン（焙煎度別色）、右上バッジ、タイトル2行固定
  - タグ表示（2行 + 溢れ時ドロップダウン）、投稿者名表示
  - ダミーデータでタグ・投稿者を表示
- [ ] タグクリック時のURLパラメータ連携
- [ ] テスト作成・品質確認

---

## Phase 2: バックエンドAPI拡張

- [ ] リスト取得APIに tags, authorName を追加
- [ ] Recipe型の正式拡張（オプショナルフラグ削除）
- [ ] タグによるフィルタリングAPI対応

---

## Phase 3: ダミーデータ → API統合

- [ ] RecipeCard をAPI返却データに切り替え
- [ ] ダミーデータユーティリティの削除
- [ ] E2Eテストで動作確認

---

## Phase 4: 他ページのトンマナ統一

- [ ] デザイントーンの共通化（背景色、フォント、カラーパレット）
- [ ] レシピ詳細ページへの適用
- [ ] その他ページへの適用（必要に応じて）
